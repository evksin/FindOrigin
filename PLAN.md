План реализации FindOrigin

1. Инициализация проекта
   - Создать Next.js проект и базовую структуру.
   - Добавить переменные окружения для токена бота и секретов.

2. Настройка webhook
   - Реализовать API-роут `POST /api/webhook`.
   - Извлечь `chat.id` и `text` из `update.message`.
   - Отправлять ответ через Telegram API `sendMessage`.
   - Возвращать `200 OK` сразу после принятия update.

3. Обработка входных данных
   - Определить тип входа: текст или ссылка на Telegram-пост.
   - Если ссылка — извлечь текст поста; если текст — использовать напрямую.

4. Извлечение ключевых фактов
   - Выделить ключевые утверждения, даты, числа, имена и ссылки.
   - Сформировать структурированное представление фактов.

5. Поиск возможных источников
   - Построить запросы к поиску по категориям: официальные сайты, новости, блоги, исследования.
   - Собрать кандидатов-источников (ссылки и метаданные).

6. Сравнение смысла с помощью AI
   - Сопоставить исходные факты с найденными кандидатами по смыслу.
   - Оценить уверенность для каждого источника.

7. Формирование ответа пользователю
   - Составить список 1–3 ссылок с оценкой уверенности.
   - Подготовить читаемое сообщение для Telegram.

8. Тестирование и деплой
   - Проверить webhook локально и на тестовом чате.
   - Задеплоить на Vercel и проверить работу с реальным webhook.
